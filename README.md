# ranger

create ranges over any type

## Quick start

```sh
gleam test  # Run the tests
gleam shell # Run an Erlang shell
```

## Installation

This package can be added to your Gleam project:

```sh
gleam add ranger
```

and its documentation can be found at <https://hexdocs.pm/ranger>.
